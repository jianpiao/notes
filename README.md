# notes

> vue 2.0 兼容pc和手机的记事本

## 配置和查看

``` bash
  下载zip安装包之后=>
# 安装依赖
npm install

# 默认加载到本地端口号 localhost:8080
npm run dev

# 进行打包 方便与发行成app
npm run build

```

欢迎start
